# Base44 Project

## Prerequisites
1. Clone the repository
2. `npm install`
3. `npm install -g base44@latest`

## Run Locally
```bash
base44 dev
```

## Frontend only
```bash
npm run dev
```

Create `.env.local`:
```
VITE_BASE44_APP_ID=your_app_id
VITE_BASE44_APP_BASE_URL=https://your-app.base44.app
```
