## What Users Hate (Churn Reasons)

### Seeking Alpha
- Too much opinion content — hard to separate signal from noise
- Sponsored articles not clearly marked → trust issue
- Paywall hits immediately, feels like bait-and-switch
- AI summaries are generic with no source citations
- Reddit signal: *"I cancelled because it's just bloggers with agendas"*

### Simply Wall St
- Scores are not explainable: *"Why does this stock have 3 snowflakes?"*
- No source attribution: *"Where does this number come from?"*
- Data lag: metrics sometimes weeks behind
- Oversimplification: nuance lost in visualizations
- Reddit signal: *"Pretty but I don't trust it for actual decisions"*

### Morningstar
- Too expensive for what retail users get (~$35/mo)
- UI feels outdated, not designed for self-directed retail
- Overwhelming complexity
- Most value locked behind analyst reports, not structured data

### Koyfin
- Steep learning curve — "feels like Bloomberg lite"
- No plain-language interpretation of data
- Designed for semi-professionals, not casual retail investors
- EU-focused investors frustrated by US-centric defaults

### Finviz
- Screener only — no narrative, no explanation layer
- Elite pricing feels high for a pure screener
- No *"what does this mean?"* layer

### Stockanalysis.com
- Great data but no intelligence layer
- No AI, no verdict, no explanation
- Users use it as a data source, not a research tool

### Finchat
- $468/yr is a high commitment for most retail investors
- No scoring, no thesis tracking
- AI answers lack source attribution

## The Blue Ocean Gap

**None of the competitors combine all four:**

1. **Plain-language verdict** — *"This stock is fairly valued because P/E of 18.4 is below sector median of 22"*
2. **Named source per metric** — *"P/E 18.4 · Source: Yahoo Finance TTM"*
3. **EU investor focus** — European stocks, EUR context, MiFID II awareness
4. **AI with citations** — not generic summaries, grounded in actual named data

**Wave 1 survey confirmation:** 71% WTP only if source attribution is present → this is the purchase trigger, not a nice-to-have.

## Pricing Defensibility for pondex

Competitive anchor points:
- Below: Stockanalysis (~$10/mo) — no intelligence layer
- Sweet spot: $9–19/mo — below Seeking Alpha + Morningstar, premium over data-only tools
- Above: Finchat ($39/mo) — AI but no attribution, no EU focus

**Positioning:** *"The only tool that shows its work"* — premium justified by trust, not data volume.

## Counterfactual

Without pondex, EU self-directed value investors continue to:
- Pay $19–35/mo for noise-heavy platforms with no source attribution
- Piece together data from 3–5 tools (fragmentation = Wave 1 Cluster 3 pain)
- Make decisions based on unverified AI summaries or opinion content
- Find no EU-focused tool combining narrative + data + citations

No existing player is moving toward source attribution or EU focus — confirmed by absence of this feature across all competitors surveyed.
