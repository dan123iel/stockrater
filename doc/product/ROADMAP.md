# Roadmap

---

## MVP — Kernidee beweisen

Ticker eingeben. Analyse verstehen. Zurückkommen.

- Score mit Erklärung pro Komponente
- Confidence Score (wie viele Datenpunkte)
- Reverse DCF ("welches Wachstum impliziert der Kurs?")
- AI-Einordnung in Alltagssprache — warum passt die Aktie zum Profil
- Data Coverage Indicator
- Watchlist
- JSON Backup/Restore

Raus: AI-Chat, Vorschläge, Thesis Tracker, Broker-Sync.

---

## V1 — Entscheidungen begleiten

- Strategie-Profil
- Portfolio (manuell + CSV)
- Thesis Tracker
- Investment Memo
- Peer-Vergleich
- Watchlist Alerts (Fair Value, Insider, Earnings)
- Score-Label: Excellent Fit / Poor Fit statt Buy / Avoid
- Bear Thesis automatisch neben Bull Case

## V2 — Proaktiv werden

- Aktienvorschläge (Profil + Portfolio + Makro)

- Earnings Translator
- Pre-Mortem Engine
- Shareable Thesis Card
